# Acunetix11-API-Documentation

Acunetix11伪API,非官方API文档。官方API需要 Enterprise edition licenses才能获取到。

## 获取API-KEY
Administrator--Profile--API Key

## 环境设置:

### Header设置:

```
X-Auth: API-KEY"
Content-type: application/json; charset=utf8
```

### 接口设置
1. 传参内容均为Json格式
2. 接口均为https
